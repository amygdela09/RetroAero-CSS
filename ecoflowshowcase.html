EcoFlow

Welcome to EcoFlow

A lightweight CSS & JS library for creating glossy, glassy, and optimistic web interfaces inspired by the Frutiger Aero aesthetic.

Get Started

Lively Background

A single JS file powers a dynamic, multi-colored bubble background that flows and interacts as you scroll.

Glass Components

Ready-to-use classes like `.feature-card` and `.cta-button` provide an instant glossy, translucent look.

Easy to Use

Simply link the CSS and JS files, add the container classes, and you're ready to start building your interface.

Live Demo: Bubble Effects

The animated background consists of multi-layered, semi-transparent orbs that bounce and blend together.

Glass & Refraction

A multi-layered design with `backdrop-filter` creates a realistic, magnifying glass effect.

Rich Colors

Orbs are randomly colored from the Frutiger Aero palette for a vibrant, optimistic feel.

Luminous Blending

Using `mix-blend-mode`, overlapping orbs blend their colors for a beautiful, liquid-light effect.

How to Use

Just add the `ecoflow-background` class to your body and include the `.bg-orbs` container div right after.

<body class="ecoflow-background">
    <div class="bg-orbs"></div>
    
    <!-- Your content here -->
    
    <script src="ecoflow.js"></script>
</body>
